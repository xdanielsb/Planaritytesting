## Configurations

### How to use it?

#### Configure  Java


```sh
$ export JAVA_HOME=/usr/local/jdk1.8.0_111
$ export PATH=$JAVA_HOME/bin:$PATH

$ javac filename.java
$ java filename
```

#### Launch the application

```sh
$ git clone https://github.com/xdanielsb/Planaritytesting-Java.git
$ cd Planaritytesting-Java/src
$ javac Launcher.java
$ java Launcher

```
